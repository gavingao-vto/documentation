VM环境

准备

要在一个虚拟机中运行有关管理中心，代理和CLI的多合一指令，您需要一个具有以下功能的虚拟机：

  • 大于等于 4GB内存
  • 20GB储存空间
  • Ubuntu Server 18.04 LTS

  • VirtualBox 6.1
